#  Bangalore House Price Prediction

This project predicts house prices based on location, square footage, BHK, and bathrooms using Machine Learning.

##  Features
- Data cleaning and preprocessing  
- Outlier removal (price_per_sqft & BHK logic)  
- ML models: Linear Regression, Ridge, Lasso  
- Flask web app for real-time predictions  
- Simple UI using HTML & Bootstrap  

##  Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Flask  
- HTML, Bootstrap  

##  How it works
1. User inputs location, BHK, bathrooms, sqft  
2. Data is processed  
3. Model predicts house price
